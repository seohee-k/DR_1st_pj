## 개요

창고 내 물품 운반 및 분류 : 소형 자율주행 로봇이 창고 내 지정된 경로를 따라 이동하여 yolo를 이용해 특정 물품이나 표지판을 인식 가능, 소규모 창고에서 저비용 자동화 구현 (ex.아마존 물류창고 Kiva robot)
